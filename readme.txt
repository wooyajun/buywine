项目所有文件都在dist文件夹中


1.首页：./dist/index.html
2.环境： ghp
	首页目录：localhost：8080
3.sass源文件：./sass/*
5.功能：
	1：注册
		用户名11位手机号
	   	密码6-16位数字、密码、下划线、中划线
	   	已有用户不能注册
	   	写错有红色字体提示
	2.登录：
		自动登录会登录最后一个用户
		账户密码不符会报错
		登录成功顶部显示用户名 
		可退出登录
	3.详情页有放大镜效果
	4.商品列表 可按照价格排序、懒加载
	5.购物车：
		不登录也可加入，结算必须先登录
		购物车中商品数量 各页面都可以看到
	6.轮播图
6.模块化、楼层未做
